# Blackjack Game (JavaScript)

This project was developed for a Data Structures course, with the goal of applying programming logic, array manipulation, and control flow concepts in an interactive game.

## Objective

To simulate the Blackjack game using JavaScript, exploring data structures to represent the deck, manage scores, and implement game rules.

## Concepts Applied

* Use of arrays to represent the deck of cards
* Shuffle algorithm implementation
* Game state management (player and dealer scores)
* Conditional structures for game rules
* DOM manipulation for dynamic rendering
* Ace handling logic (value as 1 or 11)

## Features

* Full 52-card deck
* Random card shuffling
* Automatic score calculation
* Dealer logic (draws until reaching at least 17)
* Interactive interface with buttons (Hit / Stand)
* Modal with game instructions
* Dynamic card rendering on screen

## Technologies

* HTML5
* CSS3
* JavaScript (Vanilla JS)
